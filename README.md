# Инверсия строки и столбца — мини‑игра

Правило: при клике по ячейке **инвертируются все значения** в её **строке** и **столбце** (0 ↔ 1).  
Важно: **нажатая ячейка не меняется** (она попадает и в строку, и в столбец — инвертируется дважды).  
Цель: сделать **все ячейки равными 1**.

## Как запустить

- Откройте файл `index.html` в любом современном браузере (Chrome/Safari/Firefox).

> В этой среде запуск `python3 -m http.server` может быть запрещён, но для игры это не требуется.

## Управление

- **Клик** по ячейке: сделать ход (инвертировать строку+столбец).
- **Shift + клик**: только подсветить строку/столбец.

## Возможности интерфейса

- **Строки/Столбцы**: задать размер (2×2 … 12×12).
- **Новая пустая**: заполнить поле нулями.
- **Перемешать**: сгенерировать “достижимую” позицию (через набор случайных ходов).
- **Решить**: автоматически находит и применяет решение (алгебра по mod 2) для текущих правил (вариант B).
- **Ходы (справа)**: логирует клики и шаги решателя, можно **очистить** или **скопировать** как список координат.
- **Сбросить**: вернуть стартовую матрицу.
- **Скопировать матрицу**: копирует текущую матрицу в буфер обмена.
- **Вставить/задать матрицу**: можно вставить матрицу из Excel/текста и нажать “Применить”.
